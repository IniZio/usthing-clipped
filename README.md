# usthing-thing

Things in scaffolding, dev, build, deploy

Flow:
1. Scaffolding: (eliminates need for devDependencies)
  - frontend: VueJs, ReactJs
  - backend: ServerJs
2. Dev:
  - frontend: webpack-dev-server
  - backend: nodemon / gaze
3. Build (Docker Image):
  - frontend: webpack Build
  - backend: backpack
4. Deploy
